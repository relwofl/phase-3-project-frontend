### COW Tracker
A user would be able to add a cow, get a list of cows, delete cows, edit cows (births), and add a new birth. When creating/ adding a new cow users can set name, birthdate, gender, offspring, and an optional note. 

### Features 
* Homepage
* Navigation
* All Cows Page 
* Cow details 
* Create Cow 
* Update Cow 
* Delete Cow
